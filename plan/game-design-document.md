## 1. 게임 컨셉

### 한 줄 요약
> 귀여운 토이푸들 릴리가 다양한 배경을 달리며 간식을 모으고 장애물을 피하는 횡스크롤 러닝 게임

### 컨셉 키워드
- **귀여움**: 실제 토이푸들을 기반으로 한 애니메이틱 캐릭터
- **경쾌함**: 가볍고 즐거운 플레이 경험
- **수집욕**: 간식, 코스튬, 스테이지 수집 요소
- **접근성**: 누구나 양손 엄지로 쉽게 즐길 수 있는 조작

### USP (Unique Selling Point)
> "이 게임만의 매력이 뭐야?"에 대한 명확한 답

- **밀림 시스템**: 릴리의 겁 많은 성격을 게임 메카닉에 녹임 — 피격 = 게임 오버가 아닌, 겁먹고 뒤로 밀리는 독특한 시스템
- **정서적 유대감**: 실제 강아지를 기반으로 한 캐릭터라 반려인/동물 애호가 감성 자극
- **힐링 수집**: 경쟁이 아닌 힐링/수집 중심의 러닝 게임. 귀여운 코스튬과 간식 수집에 집중

#### 경쟁작 대비 포지셔닝
```
          힐링
           ▲
           │
    달려라 릴리●
           │         ● 알토의 모험
  캐주얼 ──┼──────────── 하드코어
           │
   ● 푸시모 │   ● 쿠키런
           │
           ▼
          경쟁
```

### 세계관/스토리 설정
> 릴리가 왜 달리는지에 대한 배경

**채택 컨셉**: 꿈속에서 간식 나라를 모험하는 릴리

- 낮잠에 빠진 릴리가 꿈속 간식 나라에 떨어지면서 모험이 시작
- 각 스테이지는 꿈속 세계의 한 장소 — 현실 배경을 몽환적으로 재해석한 공간
- 간식을 모으면 다음 꿈의 문이 열리고, 새로운 세계로 이동
- 스테이지 클리어 시 짧은 컷씬(1~2컷 일러스트)으로 릴리의 반응 표현
- 스토리는 분위기 연출용으로, 게임 진행에 직접적 영향 없음 (가볍게 유지)

> **대안**: (A안) 산책 중 목줄이 풀려 모험 / (C안) 다양한 동네 산책 일상 — 필요 시 재검토

### 레퍼런스
| 작품 | 참고 요소 | 가져올 것 | 가져오지 않을 것 |
|------|-----------|-----------|-----------------|
| 쿠키런 | 횡스크롤 러닝 구조, 캐릭터 수집 | 청크 기반 맵 생성, 코스튬 수집 시스템 | 복잡한 스킬/조합 시스템, 가챠 |
| 알토의 모험 | 배경 전환, 감성적 분위기 | 파스텔톤 배경, 시간대 변화 연출 | 원버튼 조작 (너무 단순) |
| 푸시모 | 귀여운 동물 캐릭터, 아기자기한 아트 | 캐릭터 표정/리액션 디테일 | 방치형 시스템 |

## 2. 타겟 유저

### 기본 정의
| 구분 | 내용 |
|------|------|
| **주 타겟** | 10~30대 여성, 캐주얼 게임 선호 유저 |
| **부 타겟** | 반려동물(강아지) 애호가, 힐링 게임 유저 |
| **플레이 성향** | 짧은 세션(1~3분), 출퇴근/대기 시간 활용 |
| **접근성** | 게임 경험이 적은 유저도 쉽게 입문 가능 |

### 유저 페르소나

**페르소나 A — 지하철 힐링러**
- 25세 여성 직장인, 출퇴근 지하철에서 플레이
- 강아지 키우고 싶지만 여건상 못 키움
- 귀여운 캐릭터 보며 힐링, 코스튬 수집에 관심
- 과금 성향: 월 5,000원 이하, 한정 코스튬에만 소비

**페르소나 B — 수집형 학생**
- 16세 학생, 귀여운 캐릭터 수집 좋아함
- 쿠키런 유경험자, 도감 채우기에 강한 동기부여
- SNS에 게임 스크린샷 공유 즐김
- 과금 성향: 무과금, 광고 시청으로 코인 수집

**페르소나 C — 반려인 공감러**
- 32세 반려인, 실제 푸들 보호자
- "우리 강아지랑 비슷하다!" 공감 포인트로 유입
- SNS 공유 적극적, 반려동물 커뮤니티 활동
- 과금 성향: 시즌 패스 구매 의향, 한정 코스튬 관심

### 시장 포지셔닝
| 비교 항목 | 달려라 릴리 | 쿠키런 | 알토의 모험 |
|-----------|-----------|--------|------------|
| 난이도 | 쉬움 | 보통~어려움 | 쉬움 |
| 수집 요소 | 코스튬/간식/도감 | 캐릭터/펫/보물 | 없음 |
| BM 강도 | 약함 (광고+코스튬) | 강함 (가챠+패스) | 없음 (유료 앱) |
| 소셜 기능 | 랭킹/공유 | 길드/PvP/랭킹 | 없음 |
| 감성 | 힐링/귀여움 | 화려함/경쟁 | 명상/고요 |

### 마케팅 접점
- **주요 채널**: 인스타그램, 틱톡 (귀여운 릴리 숏폼 영상)
- **바이럴 요소**: 결과 화면 스크린샷 공유, 코스튬 입은 릴리 이미지, 게임 내 귀여운 순간 GIF 캡처
- **반려동물 커뮤니티**: 강아지 관련 카페/커뮤니티 타겟 광고

## 3. 핵심 루프 (Core Loop)

### 멀티 루프 구조

```
[1차 루프 - 세션 단위 (1~3분)]
달리기 → 간식 수집 & 장애물 회피 → 점수/코인 획득 → 재도전

[2차 루프 - 일 단위]
일일 미션 확인 → 플레이 → 미션 달성 → 보상 수령 → 코스튬 구매/강화

[3차 루프 - 주/월 단위]
시즌 패스 진행 → 랭킹 경쟁 → 시즌 보상 획득 → 다음 시즌 대기
```

### 1차 루프 상세 흐름
1. **달리기**: 릴리가 자동으로 달림 (속도 점진적 증가)
2. **액션**: 점프(왼쪽), 슬라이딩(오른쪽)으로 장애물 회피 & 간식 수집
3. **결과**: 달린 거리 + 수집 간식으로 점수 산출
4. **보상**: 코인 획득 → 코스튬/아이템 구매
5. **성장**: 새로운 능력/코스튬 장착 → 더 높은 점수 도전

### 코인 경제 기초 설계

```
[1회 플레이 평균 수입]
- 기본 간식: ~30개 × 1코인 = 30 코인
- 소고기: ~2개 × 5코인 = 10 코인
- 스테이지 클리어 보상: 50~200 코인 (스테이지별 차등)
- 1회 평균 합계: ~100~240 코인

[일일 수입 (하루 10회 플레이 기준)]
- 플레이 수입: ~1,000~2,400 코인
- 일일 미션 보상: ~500 코인
- 광고 시청 (3회): ~300 코인
- 일일 합계: ~1,800~3,200 코인

[지출처]
- 일반 코스튬: 500~1,000 코인
- 희귀 코스튬: 2,000~3,000 코인
- 특별 코스튬: 5,000~8,000 코인

[밸런스 목표]
- 무과금 유저: 2~3일에 일반 코스튬 1개 구매 가능
- 희귀 코스튬: 약 1주일
- 특별 코스튬: 약 2~3주
```

> 코인 인플레이션 방지: 보유 코인 상한 없음, 대신 지속적으로 새 코스튬 추가로 소비처 확보

### 진행도 시스템
- **전체 진행률** = 스테이지 진행 + 코스튬 수집 + 도감 완성
- 진행률 마일스톤 보상:
  - 25%: 한정 코스튬 (리본)
  - 50%: 한정 코스튬 (왕관)
  - 75%: 한정 코스튬 (날개)
  - 100%: 전설 코스튬 (무지개 망토)

## 4. 게임 메카닉

### 4.1 조작법

> 화면을 좌/우 영역으로 나누어 양손 엄지로 조작. 터치 영역은 화면 절반씩 크게 잡아 정확히 안 봐도 누를 수 있도록 설계.

| 입력 | 동작 | 릴리 모션 |
|------|------|-----------|
| 왼쪽 터치 (탭) | 점프 | 네 발로 폴짝 뛰기 |
| 왼쪽 길게 터치 | 높은 점프 (차지) | 웅크렸다 크게 점프 |
| 오른쪽 터치 (탭) | 슬라이딩 | 배 깔고 미끄러지기 |

> **조작 연속성**: 슬라이딩 중 점프 입력 시 슬라이딩 즉시 캔슬 → 점프 발동. 동작 간 전환에 딜레이 없이 반응하여 답답함 방지.

#### 조작 상세 스펙

| 항목 | 수치 | 비고 |
|------|------|------|
| 차지 점프 최소 홀드 시간 | 300ms | 이하 = 일반 점프 |
| 차지 점프 최대 홀드 시간 | 800ms | 이상 = 최대 높이 |
| 차지 게이지 UI | 릴리 아래 원형 게이지 | 300ms부터 표시 시작 |
| 슬라이딩 지속 시간 | 홀드하는 동안 유지 | 최대 2초, 이후 자동 해제 |
| 입력 버퍼링 | 착지 0.1초 전 입력 허용 | 착지 즉시 해당 동작 발동 |

#### 공중 상태 입력 규칙
| 현재 상태 | 입력 | 결과 |
|-----------|------|------|
| 점프 중 | 오른쪽 탭 | 입력 버퍼링 → 착지 즉시 슬라이딩 |
| 차지 중 | 피격 | 차지 캔슬 + 밀림 발동 |
| 슬라이딩 중 | 왼쪽 탭 | 즉시 캔슬 → 점프 |
| 부스트 중 | 모든 입력 | 무시 (자동 진행) |

### 4.2 장애물

#### 장애물 vs 지형 이벤트 분류
- **장애물** (피격 시 밀림): 대형견, 울타리, 간판, 자전거, 비둘기 떼
- **지형 이벤트** (밀림 없음): 눈밭 (연출 이벤트)

#### 공통 장애물
| 장애물 | 회피 방법 | 밀림 거리 | 회복 시간 | 설명 |
|--------|-----------|-----------|-----------|------|
| 다른 강아지 (대형견) | 점프/슬라이딩 | 크게 (3칸) | 1.5초 | 부딪히면 겁먹고 뒤로 밀림 |
| 자전거 | 높은 점프 (차지) | 크게 (3칸) | 1.5초 | 높이 뛰어서 넘기 |
| 낮은 울타리 | 점프 | 보통 (2칸) | 1.0초 | 기본 장애물 |
| 높은 간판 | 슬라이딩 | 보통 (2칸) | 1.0초 | 숙여서 통과 |
| 비둘기 떼 | 슬라이딩 | 적게 (1칸) | 0.5초 | 놀라서 날아감 (연출) |

#### 지형 이벤트
| 이벤트 | 효과 | 설명 |
|--------|------|------|
| 눈밭 | 속도 변화 (연출) | 밟으면 릴리가 신나서 꼬리 세우고 발 구르기 |

#### 스테이지 전용 장애물
| 스테이지 | 전용 장애물 | 회피 방법 | 밀림 | 설명 |
|----------|-----------|-----------|------|------|
| 4. 해변 | 파도 | 점프 (타이밍) | 2칸 | 주기적으로 밀려오는 파도 |
| 5. 숲속 | 낮은 나뭇가지 | 슬라이딩 | 1칸 | 높이가 낮은 나뭇가지 |
| 6. 눈마을 | 눈사람 | 점프 | 2칸 | 고정형 큰 장애물 |
| 7. 비골목 | 미끄러운 바닥 | 지형 이벤트 | - | 조작 감도 변화 (미끄러짐) |
| 8. 놀이공원 | 움직이는 풍선 | 점프/슬라이딩 | 2칸 | 상하 이동하는 장애물 |
| 9. 캠핑장 | 어둠 구간 | 랜턴 아이템 | - | 시야 제한, 랜턴으로 확보 |

> **피격 시스템**: 게임 오버가 아닌 **뒤로 밀림** 방식. 릴리가 겁먹는 모션과 함께 해당 칸 수만큼 후퇴. 밀려난 위치가 누적되어 **화면 왼쪽 경계 밖**(릴리 기본 위치에서 약 8칸 후퇴)으로 밀려나면 게임 오버.

> **장애물 예고 신호**: 장애물 등장 1.5초 전, 화면 오른쪽 가장자리에 해당 장애물의 그림자/실루엣 표시. 대형견은 추가로 느낌표(!) 아이콘 표시.

### 4.3 수집 아이템

| 아이템 | 효과 | 지속 시간 | 출현 확률 | 점수 | 비고 |
|--------|------|-----------|-----------|------|------|
| 강아지 간식 (기본) | 기본 점수 획득 | - | 매우 높음 | 10점 | 기본 배치, 라인 형태 |
| 소고기 (희귀) | 높은 점수 + 코인 | - | 낮음 | 100점 + 코인 5 | 스테이지 중후반 등장 |
| 솔방울 | 속도 소폭 증가 + 밀림 내성 | 5초 | 보통 | - | 속도 1.3배, 무적 아님 |
| 자석 | 간식 자동 흡수 | 7초 | 낮음 | - | 흡수 범위 화면 1/3 |
| 방패 (목줄) | 장애물 1회 무시 | 1회 사용 | 매우 낮음 | - | 밀림 1회 완전 무시 |
| 스피드 부스트 | 무적 + 빠른 질주 | 4초 | 매우 낮음 | - | 속도 2배, 간식 자동 획득 |

#### 솔방울 vs 스피드 부스트 차이
| 비교 | 솔방울 | 스피드 부스트 |
|------|--------|-------------|
| 속도 | 1.3배 (소폭) | 2배 (대폭) |
| 무적 여부 | 아님 (밀림 내성만) | 완전 무적 |
| 간식 흡수 | 수동 수집 | 지나가는 간식 자동 획득 |
| 연출 | 입에 물고 씩씩하게 달리기 | 집중선 + 무지개 이펙트 |

> **보너스 스테이지**: 간식 **50개** 수집 시 보너스 스테이지 진입. 장애물 없이 간식을 최대한 수집하는 구간 (5초간). 수집량에 따라 추가 코인 보상.

### 4.4 스테이지 & 난이도 설계

> 10개의 스테이지를 순서대로 클리어하는 구조. 스테이지가 올라갈수록 속도가 빨라지고 장애물이 복잡해짐.

| 스테이지 | 배경 | 속도 | 특징 |
|----------|------|------|------|

### 4.5 맵 생성 방식 (청크 시스템)

> 완전 랜덤 배치 금지. 패턴 기반 청크(Chunk) 조합으로 공정한 난이도 보장.

- 장애물+간식이 묶인 **청크 단위**를 사전 설계 (스테이지당 20~30개 패턴)
- 진행 거리/스테이지에 따라 등장 청크 풀(pool)이 변경
- 모든 청크는 **회피 가능성 검증** 완료 후 투입 (넘을 수 없는 구간 원천 차단)

```
[청크A: 울타리+간식3개] → [청크B: 대형견+소고기] → [청크C: 간판+비둘기+간식5개] → ...
```

| 구간 | 청크 구성 |
|------|-----------|
| 초반 | 단일 장애물 + 간식 다수 |
| 중반 | 2연속 장애물 + 아이템 배치 |
| 후반 | 3연속 복합 장애물 + 희귀 아이템 |

#### 청크 샘플 (스테이지 1 기준)

| 청크 ID | 구성 | 난이도 | 회피 방법 |
|---------|------|--------|-----------|
| 1-A | 간식 5개 라인 (직선) | 쉬움 | 그냥 달리기 |
| 1-B | 울타리 1개 + 간식 3개 (위) | 쉬움 | 점프 |
| 1-C | 간판 1개 + 간식 3개 (아래) | 쉬움 | 슬라이딩 |
| 1-D | 울타리 → 간판 연속 + 간식 2개 | 보통 | 점프 → 슬라이딩 |
| 1-E | 대형견 1마리 + 소고기 1개 (뒤) | 보통 | 점프/슬라이딩 |

#### 청크 규칙
- 청크 간 **최소 간격**: 3m (빈 구간)으로 유저에게 숨 돌릴 틈 제공
- 동일 청크 **연속 등장 방지**: 직전 2개와 다른 청크 선택
- 난이도별 청크 풀 비율:

| 구간 (거리 기준) | 쉬움 | 보통 | 어려움 |
|----------------|------|------|--------|
| 0~30% | 70% | 30% | 0% |
| 30~70% | 30% | 50% | 20% |
| 70~100% | 10% | 40% | 50% |

### 4.6 난이도 커브 (Tension Design)

> 단순 선형 속도 증가가 아닌, 긴장과 이완을 반복하는 파동형 설계

```
텐션
 ▲
 │    ╱╲      ╱╲        ╱╲
 │   ╱  ╲    ╱  ╲      ╱  ╲      ╱ ← 최종 구간
 │  ╱    ╲  ╱    ╲    ╱    ╲    ╱
 │ ╱      ╲╱  보너스 ╲╱  보너스 ╲╱
 │╱
 └──────────────────────────────→ 진행 거리
```

- **긴장 구간**: 속도 증가 + 장애물 밀도 상승
- **이완 구간**: 보너스 스테이지, 간식 러시, 연출 이벤트 (눈밭 등)
- 스테이지 중반마다 **미니 피버 타임** 삽입으로 쾌감 제공

#### 난이도 수치
| 항목 | 수치 |
|------|------|
| 스테이지 내 속도 증가 | 10m마다 기본 속도의 1% 증가 (최대 속도까지) |
| 긴장:이완 거리 비율 | 7:3 |
| 피버 타임 발동 조건 | 피격 없이 200m 연속 달성 시 |
| 피버 타임 지속 시간 | 5초 (무적 + 간식 2배 점수) |

- 전경/중경/후경 3레이어 패럴랙스 스크롤로 입체감 연출
- 스테이지 내 시간대 변화: 아침 → 낮 → 저녁 (진행도에 따라)

#### 배경 오브젝트 예시
| 스테이지 | 후경 | 중경 | 전경 | 동적 오브젝트 |
|----------|------|------|------|-------------|

| 2. 벚꽃길 | 분홍 하늘 | 벚꽃나무 | 꽃잎 깔린 길 | 벚꽃잎 날림 |

> 배경 오브젝트는 순수 장식. 터치 상호작용 없음 (조작 혼동 방지).

### 6.3 스테이지 해금 및 전환
- 이전 스테이지 클리어 시 다음 스테이지 해금
- 별 수집 누적에 따라 히든 보너스 스테이지 등장 가능
- 계절 이벤트 시 한정 스테이지 추가

#### 스테이지 전환 연출
- **클리어 시**: 별 1~3개가 차례로 떨어지며 획득 애니메이션 → 다음 스테이지 경로 해금 연출
- **스테이지 선택 화면**: 산책로 형태의 맵 — 릴리가 각 스테이지 위치에 서있고, 터치 시 해당 배경 미리보기
- **잠긴 스테이지**: 자물쇠 아이콘 + 실루엣 배경으로 호기심 유발

- 보상형 광고: 일일 시청 제한 10회
- 플레이 중 광고 노출 절대 금지
- 광고 제거 상품 구매 시 배너/전면 완전 제거 (보상형은 선택 시청 유지)

### 9.4 BM 원칙
- **Pay to Win 금지**: 과금은 꾸미기/편의 요소에만 적용

- **과도한 광고 지양**: 플레이 흐름을 끊지 않는 선에서 배치
- **확률형 아이템 없음**: 코스튬 뽑기는 결과 확인 가능한 선택형 (가챠 X)

#### 엔진 선택: Unity (확정)
- 모바일 빌드 안정성, 에셋 스토어, 레퍼런스(쿠키런 등) 풍부
- 완성도 높은 모바일 게임에 가장 검증된 엔진
- 언어: C#

### 10.2 성능 최적화

#### 오브젝트 풀링 (Object Pooling) - 필수
- 배경, 장애물, 간식 등 반복 생성/파괴되는 오브젝트는 **사전 생성 후 재사용**
- 매 프레임 Instantiate/Destroy 호출 금지 → GC 스파이크로 인한 프레임 드랍 방지
- 풀링 대상: 간식, 장애물, 파티클 이펙트, 배경 타일

#### 충돌 판정 (Hitbox) 최적화
- **릴리 피격 판정**: 실제 그래픽보다 **약간 작게** (관대한 회피)
- **간식 획득 판정**: 실제 그래픽보다 **약간 크게** (자석 없이도 수집감 좋게)
- 원형/박스형 단순 콜라이더 사용으로 연산량 절감
- 픽셀 단위 정밀 충돌 사용 금지

#### 패럴랙스 스크롤링 (Parallax Scrolling)
- 4레이어 구성: 하늘(원경) → 배경(중경) → 오브젝트(근경) → 바닥
- 레이어별 이동 속도 차등 적용으로 깊이감 연출

| 레이어 | 속도 비율 | 예시 |
|--------|-----------|------|
| 원경 (하늘/구름) | 0.1x ~ 0.2x | 구름, 산, 하늘 |
| 중경 (배경) | 0.4x ~ 0.6x | 건물, 나무, 원경 오브젝트 |
| 근경 (오브젝트) | 1.0x | 장애물, 간식, 릴리 |
| 바닥 | 1.0x | 지면 타일 |

## 12. 리텐션 (유지율) 설계

### 12.1 일일 미션

> 매일 갱신되는 미션으로 단기 목표와 보상 제공. 매일 3~5개 표시, 전체 풀 20개 이상.

| 미션 예시 | 난이도 | 보상 |
|-----------|--------|------|
| 슬라이딩 50회 성공하기 | 쉬움 | 코인 200 |
| 소고기 10개 수집하기 | 보통 | 코인 300 |
| 스테이지 3 별 3개로 클리어 | 어려움 | 코인 500 |
| 솔방울 물고 500m 달리기 | 보통 | 희귀 간식 상자 |
| 피격 없이 1000m 달리기 | 어려움 | 코스튬 뽑기권 |
| 점프 100회 하기 | 쉬움 | 코인 150 |
| 보너스 스테이지 3회 진입 | 보통 | 코인 300 |
| 간식 200개 수집하기 | 쉬움 | 코인 200 |
| 부스트 아이템 5회 사용 | 보통 | 코인 250 |
| 방패(목줄) 아이템 3회 획득 | 어려움 | 코인 400 |

> **주간 미션**: 매주 월요일 갱신, 3개 고정. 일일 미션보다 큰 보상 (코인 1,000~2,000 또는 희귀 코스튬).

### 12.2 출석 보상
- 연속 출석 시 점진적 보상 증가
- 7일 연속 출석 시 한정 코스튬 지급
- 28일(1시즌) 개근 시 특별 코스튬

### 12.3 도감 수집
- 스테이지별 숨겨진 수집 요소 (각 스테이지 특산 간식)
- 도감 완성도에 따라 칭호/보상 해금

### 12.4 시즌제 운영
- 시즌마다 신규 이벤트 스테이지 + 한정 코스튬 추가
- 시즌 패스를 통한 단계별 보상 체계

### 12.5 랭킹 시스템
| 항목 | 설계 |
|------|------|
| 랭킹 종류 | 스테이지별 최고 거리, 주간 총 간식 수집량 |
| 갱신 주기 | 주간 리셋 (매주 월요일) |
| 보상 | 상위 10%: 한정 코스튬 / 상위 30%: 코인 1,000 / 참여: 코인 200 |
| 친구 랭킹 | 소셜 연동 시 친구 간 순위 비교 |

### 12.6 소셜 기능
- **친구 초대**: 초대 링크로 친구 가입 시 양쪽 코인 500 지급
- **SNS 공유**: 결과 화면에 공유 버튼 → 릴리 + 점수가 담긴 카드 이미지 생성
- **공유 보상**: 공유 1회당 코인 50 (일일 3회 제한)

### 12.7 푸시 알림 전략
| 알림 종류 | 발송 조건 | 기본 설정 |
|-----------|-----------|-----------|
| 일일 미션 리마인더 | 오후 6시, 미달성 미션 있을 때 | ON |
| 새 시즌/이벤트 | 시즌/이벤트 시작 시 | ON |
| 복귀 유도 | 3일 미접속 시 | ON |
| 친구 활동 | 친구가 내 기록 갱신 시 | OFF |

- 알림 빈도 상한: **하루 최대 2회**
- 유저가 알림 종류별 개별 on/off 가능 (설정 화면)
